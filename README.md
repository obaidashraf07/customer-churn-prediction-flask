#** Customer Churn Prediction | Data Science Project**

## Project Overview

This project is an end-to-end Data Science project that predicts whether a customer is likely to leave a company based on customer demographics, subscription details, and service usage.

The project follows the complete Data Science lifecycle, from data preprocessing and exploratory data analysis (EDA) to machine learning model development and deployment using Flask.

---

## Project Objectives

- Analyze customer behavior
- Clean and preprocess data
- Perform Exploratory Data Analysis (EDA)
- Build and compare multiple machine learning models
- Select the best-performing model
- Deploy the trained model as a Flask web application

# Development Workflow

This project was developed using both **Jupyter Notebook** and **Visual Studio Code**, as each tool serves a different purpose in a real-world Data Science workflow.

---

## Work Completed in Jupyter Notebook

Jupyter Notebook was used during the Data Science phase of the project because it provides an interactive environment for experimentation and analysis.

### Tasks Completed

- Data Loading
- Data Cleaning
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Feature Selection
- Model Training
- Model Comparison
- Model Evaluation
- Saving the Best Model using Joblib

### Why Jupyter Notebook?

Jupyter Notebook allows developers and data scientists to execute code step by step, visualize results instantly, and experiment with different machine learning techniques efficiently. It is widely used for research, data analysis, and model development.

---

## Work Completed in Visual Studio Code

Visual Studio Code was used after the machine learning model was finalized to build the complete application.

### Tasks Completed

- Flask Backend Development
- Loading the Trained Model
- Creating Prediction API
- HTML Form Development
- CSS Styling
- Handling User Inputs
- Displaying Prediction Results
- Debugging
- Project Folder Organization
- Git & GitHub Integration

### Why Visual Studio Code?

Visual Studio Code is better suited for software development. It provides a professional environment for writing Python applications, developing Flask projects, managing files, debugging code, and integrating version control with Git and GitHub.
